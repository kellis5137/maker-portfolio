# Maker Portfolio — CAD, 3D Printing & Metalwork

### 👉 View the live portfolio: **https://kellis5137.github.io/maker-portfolio/**

This repository holds the source for the portfolio site along with the
underlying project files (FreeCAD designs, STL models, gcode, etc.).

If you landed here looking for the showcase, head to the link above.

## Repository layout

- [`portfolio/`](portfolio/) — Vite + React portfolio site (published to GitHub Pages)
- `projects/` — CAD source, 3D models, gcode and related project files

## Running the portfolio locally

```bash
cd portfolio
npm install
npm run dev
```

To publish updates to GitHub Pages:

```bash
cd portfolio
npm run deploy
```
